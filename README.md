# Machine Learning Journey 🧠

A compilation of files that explain key machine learning principles that I am learning.

This repo documents my 52-week project-based journey to becoming a machine learning 
scientist and engineer — covering everything from the mathematical foundations to 
building and researching original algorithms.

---

## Structure

Each folder contains the project, code, and notes for that week.

| Week | Topic | Project |
|------|-------|---------|
| 01 | Git & GitHub | Learning journal setup |
| 02 | Python for Data Science | EDA on a public dataset |
| 03 | Statistics & Probability | Statistical report from scratch |
| 04 | Linear Algebra | Matrix operations library |
| 05 | Calculus & Optimization | Gradient descent from scratch |
| 06 | Data Cleaning & Pipelines | Messy dataset → clean pipeline |
| 07 | Data Visualization | Interactive dashboard |
| 08 | Phase 1 Capstone | End-to-end data analysis |

*Table will be updated weekly as new projects are added.*

---

## Phases

- **Phase 1 (Weeks 1–8):** Foundations — Python, math, data skills
- **Phase 2 (Weeks 9–18):** Core ML — implementing algorithms from scratch
- **Phase 3 (Weeks 19–28):** Deep Learning — CNNs, transformers, LLMs
- **Phase 4 (Weeks 29–40):** Specialization — RL, diffusion models, MLOps
- **Phase 5 (Weeks 41–52):** Research — reading papers, designing new algorithms

---

## Goals

- Build every algorithm from scratch before using libraries
- Ship one project every week, every week
- Learn to read and reproduce ML research papers
- Contribute to open source by week 49
- Write and publish an original research paper by week 52
